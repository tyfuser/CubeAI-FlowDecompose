# 🚀 快速参考 - 镜头拆解分析

## 📍 功能位置

侧边栏 → 🎬 **镜头拆解分析**

---

## 🎯 核心功能

| 功能 | 说明 |
|-----|------|
| 视频上传 | 支持拖拽或路径输入 |
| 自动分析 | AI 识别镜头、运镜、光线、调色 |
| 时间轴 | 可视化展示所有镜头和特征 |
| 详情查看 | 点击片段查看详细分析 |
| 历史记录 | 查看和管理分析历史 |

## 🎨 时间轴说明

```
┌─────────────────────────────────────┐
│ 00:00    00:10    00:20    00:30   │ ← 时间标尺
├─────────────────────────────────────┤
│ [seg_01][seg_02][seg_03][seg_04]   │ ← 镜头片段
├─────────────────────────────────────┤
│ [特写][中景][推镜][全景]            │ ← 运镜
├─────────────────────────────────────┤
│ [三点布光][自然光][背光]            │ ← 光线
├─────────────────────────────────────┤
│ [暖色调][冷色调][高对比]            │ ← 调色
└─────────────────────────────────────┘
```

## 🔍 特征识别

### 运镜类型
- 特写 (Close Up)
- 中景 (Medium Shot)
- 全景 (Wide Shot)
- 推镜 (Zoom In)
- 拉镜 (Zoom Out)
- 左/右摇 (Pan)
- 跟镜 (Tracking)

### 光线类型
- 三点布光
- 自然光
- 背光
- 高调光
- 低调光

### 调色风格
- 暖色调
- 冷色调
- 高对比
- 低对比
- 鲜艳
- 去饱和

## ⚙️ 配置要求

### 后端 API
```env
VITE_SHOT_ANALYSIS_BASE_URL=http://localhost:8000
```

### 接口路径
- 创建任务: `POST /v1/video-analysis/jobs`
- 查询状态: `GET /v1/video-analysis/jobs/{job_id}`
- 历史记录: `GET /v1/video-analysis/history`

---

## 🐛 常见问题

### 无法连接后端
✅ 检查后端服务是否启动  
✅ 确认端口是 8000  
✅ 查看 `.env` 配置

### 分析时间过长
✅ 正常情况 1-2 分钟  
✅ 视频越长分析越慢  
✅ 可在历史记录查看结果

### 识别不准确
✅ 查看置信度分数  
✅ 选择高质量视频  
✅ 避免过度剪辑的视频
